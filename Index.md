## Cài đặt django
```
pip install Django
```

## Tạo project
```
django-admin startproject mysite
```
#### Trong đó:
**mysite**: Tên project


## Chạy server
#### Chuyển đến thư mục có chứa file manage.py
Sau đó dùng lệnh:
```
python manage.py runserver
```

## Tạo Web App
```
python manage.py startapp polls
```
#### Trong đó:
**polls**: Tên app
